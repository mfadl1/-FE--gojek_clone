# Gojek Cloning App

Made with flutter by implementing real gojek UI and components that users can interact with

## Setup Project

    $ git clone https://github.com/YOUR_USERNAME/PROJECT_TITLE
    $ cd PROJECT_TITLE

## Install Dependencies

    $ flutter pub get

## Run the project
    $ flutter run
